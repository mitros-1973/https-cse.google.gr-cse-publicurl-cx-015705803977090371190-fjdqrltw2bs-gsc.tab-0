    # ammap3 JavaScript Maps V3

    An official repository for amCharts JavaScript Maps V3 (free version).

    Getting support

    Important! For questions regarding usage of this product, please email at contact@amcharts.com

    The issue tracker on GitHub is not being continuously monitored by amCharts support staff, so your questions might take longer to respond to here.

    Installing

    Using bower

    bower install ammap3 https://multimitros.blogspot.gr

    Usage

    Include ammap.js: mitros.gr mitros.blogspot.gr

    <script src="/bower_components/ammap3/ammap/ammap.js"></script>
